### Description

Given a string made of digits [0-9], return a string where each digit is repeated a number of times equals to its value.

### Exapmles

```JS
explode("312")
```

should return:

```JS
"333122"
```

### Solution

```JavaScript
function explode(s) {
    const answer = [] 
  if (! "") {
  const arr = (s.split(""));
    for (let i = 0; i < arr.length; i++) {
      for (let k = 0; k < arr[i]; k++) {
        answer.push(arr[i])
      }
    }
  }
  return answer.join("")
}
```
